= MetroCluster Tiebreaker-Konfigurations-Workflow
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach der Installation der Tiebreaker-Software können Sie MetroCluster-Konfigurationen hinzufügen, SNMP-Einstellungen konfigurieren und mit der Überwachung Ihrer MetroCluster-Umgebung beginnen.

.Informationen zu diesem Workflow
Sie können diesen Workflow verwenden, um die Tiebreaker-Software nach der Installation zu konfigurieren.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Eins"]link:concept_configuring_the_tiebreaker_software.html["Konfigurieren der Tiebreaker Software"]
[role="quick-margin-para"]
Fügen Sie MetroCluster-Konfigurationen zur Tiebreaker-Software hinzu, importieren Sie Zertifikate und konfigurieren Sie die Überwachungseinstellungen.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Zwei"]link:task_configure_snmp_setts_for_tiebreaker_software.html["Konfigurieren Sie die SNMP-Einstellungen für MetroCluster Tiebreaker-Software"]
[role="quick-margin-para"]
Konfigurieren Sie die SNMP-Einstellungen so, dass die Tiebreaker-Software SNMP-Benachrichtigungen für MetroCluster-Ereignisse senden kann.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Drittens"]link:concept_monitoring_the_mcc_configuration.html["Überwachen Sie die MetroCluster-Konfiguration"]
[role="quick-margin-para"]
Mit der Tiebreaker-Software können Sie den MetroCluster-Konfigurationsstatus überwachen, SNMP-Ereignisse anzeigen und AutoSupport-Einstellungen verwalten.
