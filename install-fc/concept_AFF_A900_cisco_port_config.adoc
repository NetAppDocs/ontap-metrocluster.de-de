= Verwendung von Cisco Ports für AFF A900 oder FAS9500 Controller in einer MetroCluster Konfiguration mit ONTAP 9.10.1 oder höher

:icons: font
:imagesdir: ../media/

[NOTE]

* MetroCluster 2 oder DR 2 wird von 9132T-Switches nicht unterstützt.
