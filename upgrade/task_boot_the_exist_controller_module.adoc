= Booten des vorhandenen Controller-Moduls
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie müssen das vorhandene Controller-Modul bei ONTAP booten.

.Schritte
. Booten Sie an der LOADER-Eingabeaufforderung das vorhandene Controller-Modul in ONTAP:
+
`boot_ontap`
